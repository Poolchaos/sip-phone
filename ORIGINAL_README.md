# ZaiCommunicator

## Usage

### Clone the app

1.  Clone repo and run `npm install`
1.  Run app with `au run`
1.  Load `dist` folder as an unpacked extension in chrome://extensions

### Build the app

You can run the following command:
\$`npm build` or \$`au build`

### Start the app

You can run the following command:
\$`npm start` or \$`au run`
